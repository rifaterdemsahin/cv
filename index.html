🗺️ Project Index

Welcome to the CV project! This index will help you navigate the most important resources in this repository.

🚀 Core Files

  • 📄 README.md: The main README file for the project, containing a general overview and links to key resources.

  • 🤖 gemini.md: The project requirements and technology stack.

  • 🧭 cursor.md: An explanation of the CV management system and its objectives.

📂 CV Versions

This project contains several versions of the CV, tailored for different roles and industries.

  • v19 - HPC Engineer: The latest version, focused on High-Performance Computing.

  • v17 - AI Engineer: A version tailored for AI Engineering roles.

  • v16 - DevOps Engineer: A version focused on DevOps and Contract Architecture.

  • ⚛️ React Application: The source code for the React-based CV viewer.

  • 📊 CV Data: The core data for the CV, including experiences, skills, and contact information in JSON format.

  • ✍️ Markdown CV: The CV content broken down into individual Markdown files.
